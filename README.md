# 📊 Global Superstore Dashboard

## 🎯 Objective
Build an interactive Streamlit dashboard for analyzing sales, profit, and customer performance.

## 🧠 Approach
- Loaded and cleaned dataset
- Added filters (Region, Category, Sub-Category)
- Created KPIs and visual charts using Plotly
- Built interactive Streamlit dashboard

## 📊 Features
- Total Sales & Profit KPIs
- Sales by Category
- Profit by Region
- Top 5 Customers
- Interactive filters

## 🚀 Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## 📌 How to Run
streamlit run app.py
