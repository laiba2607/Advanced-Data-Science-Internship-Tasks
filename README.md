# 📊 Global Superstore Dashboard

## 🎯 Objective
Build an interactive Streamlit dashboard for analyzing sales, profit, and customer performance.

## 🧠 Approach

- Added filters (Region, Category, Sub-Category)
- Created KPIs and visual charts using Plotly

## 📊 Features

- Sales by Category
- Profit by Region
- Top 5 Customers
- Interactive filters

## 🚀 Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
